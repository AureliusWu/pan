# 盘中宝

个人基金盘中估值监控 PWA，面向手机端使用，专注于盘中估值、指数行情、持仓跟踪与多设备同步。

本项目作为独立项目维护，不再定位为蜉蝣基金测试版。

## 项目定位

盘中宝是一个轻量、移动优先的基金盘中观察工具。

它的目标不是做复杂交易系统，而是帮助用户在手机端快速查看：

- 当前持仓基金的盘中估值变化
- 主要指数与黄金行情
- 基金持仓市值与涨跌情况
- 关注基金的实时表现
- 多设备之间的持仓同步

## 功能方向

- 📈 基金盘中估值监控
- 🌍 顶部指数行情栏
- 🧮 持仓市值与盈亏计算
- 👀 关注模式：可添加 0 份额基金，只观察估值
- ☁️ GitHub Gist 云同步
- 📱 PWA 支持，可添加到手机主屏
- 🔄 下拉刷新与自动刷新
- 🎨 青绿插金视觉主题

## 技术结构

纯前端零框架单页应用：

- `index.html` — 页面结构与 PWA 入口
- `css/style.css` — 青绿插金主题样式
- `js/app.js` — 业务逻辑、估值获取、持仓管理、云同步
- `manifest.json` — PWA 配置
- `sw.js` — Service Worker 离线缓存与更新策略

## 项目关系

- `pan`：盘中宝，独立维护的基金盘中估值工具
- `FundVal`：蜉蝣基金，另一个独立基金项目
- `fund-compass`：司南基金，基金投资辅助与择时分析工具

## 部署

推送 `main` 分支后，通过 GitHub Pages 部署。
