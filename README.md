# 盘中宝

盘中宝是一个移动端优先的个人基金盘中估值监控 PWA，专注盘中估值、指数/黄金行情、持仓跟踪和多设备同步。

本项目独立维护，不是蜉蝣基金测试版。

## 功能

- 基金盘中估值监控。
- 持仓市值、今日盈亏、累计盈亏。
- 0 份额关注模式。
- 顶部指数与黄金行情栏。
- GitHub Gist 云同步。
- JSON 导入导出。
- 下拉刷新、自动刷新、通知检查。
- PWA 安装与离线缓存。
- 青绿插金视觉主题。

## 技术结构

纯前端零框架单页应用：

- `index.html`：页面结构与入口。
- `js/app.js`：业务逻辑。
- `css/style.css`：主题样式。
- `manifest.json`：PWA 配置。
- `sw.js`：Service Worker。

## 数据与存储

- 天天基金 JSONP：基金估值。
- 东方财富：备选净值与基金信息。
- 腾讯行情：指数行情。
- GitHub Gist：云端持仓同步。
- localStorage key 使用 `panzhongbao_` 前缀。

## 开发

无 npm 依赖。常用检查：

```bash
node --check js/app.js
```

推送 `main` 后通过 GitHub Pages 发布。

## 项目关系

- `pan`：盘中宝，本项目。
- `FundVal`：蜉蝣基金，独立基金盘中估值工具。
- `fund-compass`：司南基金，基金选基、择时、资产分析工具。

## 风险提示

所有估值、通知和模型数据仅供个人参考，不构成投资建议。
